# 高校科研工作管理系统 java1102

## 项目概述

高校科研工作管理系统是基于SpringBoot和Vue开发的，集成了教师管理、科研项目管理和论文成果管理等模块，致力于为教育科研领域提供全面而高效的工作流程。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：Idea
- 运行环境：JDK，Node.js

## 功能模块

系统主要包括以下功能模块：

- 教师管理
- 科研项目管理
- 论文成果管理

1. 准备运行环境：安装JDK、MySQL和Node.js
